Questa versione di BDTRE-OSM è adatta ad essere caricata su molti dispositivi GPS Garmin.


>>>>>>>>>>>>>>INSTALLAZIONE SUL DISPOSITIVO GPS<<<<<<<<<<<<<<<<

1 Estrarre la cartella Bdtre-osm-GPS.... contenuta nel file zip sul computer.

2 Creare Una cartella chiamata Garmin in via preferenziale sulla microsd del GPS, se il dispositivo non ne è provvisto utilizzare la memoria interna

3 Copiare il file gmapsupp.img dentro la cartella garmin (attenzione, se all'interno della cartella esiste già il file gmapsupp.img, questo verrà sovrascritto. Conviene fare una copia del file presente sulla sceda di memoria prima di sovrascriverlo.

4 Rimuovere la microsd dal computer ed inserirla nel GPS

5 Accendere il gps e la mappa verrà caricata


Questa mappa ed i prossimi aggiornamenti si possono scaricare da piemontegps.altervista.org
